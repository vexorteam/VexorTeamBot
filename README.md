# Бот команди Vexor
### [🇬🇧 English readme version](./README-en.md)

Telegram-бот для студії Vexor. Приймає заявки на проєкти, відповідає на часті запитання та форвардить повідомлення команді.

## Запуск

```bash
cp .env.example .env
# заповни BOT_TOKEN і TEAM_CHAT_ID
npm install
npm prod
```

## .env
```bash
BOT_TOKEN=     # токен від @BotFather
TEAM_CHAT_ID=  # ID групи або каналу
```
